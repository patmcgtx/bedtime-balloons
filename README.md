My second iOS app, circa 2014: **Bedtime Balloons**, a gamified kids' bedtime app with interactive animations, timers, and background notifications.

Built before SpriteKit existed, this project uses **cocos2d** for sprite animation and game-style interactions inside a UIKit-based iOS app.

## About

- More info: [Round Trip Software – Bedtime Balloons](https://roundtripsoftware.wordpress.com/bedtime-balloons/)
- Demo video: [Bedtime Balloons demo](https://youtu.be/P_HLeBcKXtE?si=k36kmfskB4IFzfiu)

## Technical details

- **Primary language:** Objective-C (legacy iOS era, pre-Swift)
- **Rendering/animation:** cocos2d (pre-SpriteKit generation)
- **Audio:** AVFoundation + CocosDenshion integration
- **Timing/background behavior:** custom timer flow with local notifications
- **Persistence:** Core Data for user customization/state
- **App architecture:** UIKit + sprite-driven interactive layers
